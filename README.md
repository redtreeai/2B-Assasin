#规则说明

    本游戏没有等级概念，一切靠打怪赚钱换装备，配合一定的推理找出关键人物并击杀。
    在到达夜鹰公会之前属于新手任务，到达公会后将获得四个主线任务，完成四个主线任务后才
    可能出现最终BOSS，游戏中的怪在你装备成型，技能完全的时候是没有难度的，即使看不到血条，
    怪物的血条也不会超过9999.
    
    然后过关的关键并不是打不打的过，而是找不找的到，并且触发关键的事件或者道具，才能将
    敌人击杀，一旦发现迷路了或者很久没进展，或者干不过，请果断读档。
    
    最后祝游戏愉快。
